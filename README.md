Smart Study Planner 

 Overview

Smart Study Planner is a command-line Python application that helps users organize and manage their daily tasks efficiently. It allows users to add, view, and delete tasks while also providing time-based reminders.

This project is developed as part of a Bring Your Own Project (BYOP) capstone to demonstrate real-world problem solving using Python.

⸻

 Problem Statement

Managing study schedules and daily tasks can be difficult for students. Many forget important tasks due to lack of proper planning tools.

⸻

Solution

This application provides a simple CLI-based solution where users can:
	•	Add tasks with date and time
	•	View all tasks
	•	Delete completed or unnecessary tasks
	•	Receive reminders when tasks are due

⸻

 Features
	•	 Add tasks with date & time
	•	 View saved tasks
	•	 Delete tasks
	•	 Reminder system
	•	 Data persistence using JSON file (works.json)
